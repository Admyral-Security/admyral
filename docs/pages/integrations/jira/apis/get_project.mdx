# Get Project

Get information about a Jira project. **This is useful for getting the project ID.**

<Callout type="info">
	For more information on the API, see [Get
	Project](https://developer.atlassian.com/cloud/jira/platform/rest/v3/api-group-projects/#api-rest-api-3-project-projectidorkey-get).
</Callout>

**SDK Import:**

```python
from admyral.actions import get_jira_project
```

## Arguments:

| Argument Name                             | Description                       | Required |
| ----------------------------------------- | --------------------------------- | :------: |
| **Project ID or Key** `project_id_or_key` | The ID or the key of the project. |   Yes    |

## Returns

A JSON object.

## Required Secrets

| Secret Placeholder | Description                                            |
| ------------------ | ------------------------------------------------------ |
| `JIRA_SECRET`      | Jira secret. See [Jira setup](/integrations/jira/jira) |

## SDK Example

```python
get_jira_project(
    project_id_or_key="SJ",
    secrets={"JIRA_SECRET": "my_stored_jira_secret"}
)
```
