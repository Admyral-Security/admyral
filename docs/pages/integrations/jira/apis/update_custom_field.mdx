# Update Custom Field

Updates a custom field.

<Callout type="info">
	For Jira's documentation, see [Update Custom
	Field](https://developer.atlassian.com/cloud/jira/platform/rest/v3/api-group-issue-fields/#api-rest-api-3-field-fieldid-put).
</Callout>

## Parameters:

| Field           | Description                                                                                       | Required |
| --------------- | ------------------------------------------------------------------------------------------------- | :------: |
| **Field ID**    | The ID of the custom field to update.                                                             |   Yes    |
| **Name**        | The new name of the custom field.                                                                 |   Yes    |
| **Description** | The new description of the custom field.                                                          |    No    |
| **SearcherKey** | The searcher to use for the custom field. This is required for fields of type `Text` or `Number`. |    No    |

## Example Output:

No content is returned
