# Jira

1. In **Jira**, click the Settings icon in the top right corner. Next, click on **Atlassian account settings**.

<img src="/jira_software1.png" />

2. Click on the **Security** tab. Next, click on **Create and manage API tokens**.

<img src="/jira_software2.png" />

3. Give your token a label, e.g. `Admyral`, and click **Create**. Copy the token and save it in a secure place.

<img src="/jira_software3.png" />

4. Go to the **Settings** page in **Admyral** and click on **Add New Secret** in the **Secrets** section.

5. Give your secret a name, e.g. `AlienVault`.

6. The following secrets structure is expected:

| Key       | Value                                                                                                                                                           |
| --------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `api_key` | Paste the API key here                                                                                                                                          |
| `domain`  | Paste your Jira/Atlassian domain here, e.g., if you access Jira via URL like `https://admyral.atlassian.net`, then your domain would be `admyral.atlassian.net` |
| `email`   | The email of the account who provisioned the API key                                                                                                            |

7. Click on the **Save** button.

Alternatively, you can use the following CLI command structure:

```bash
admyral secret set jira_secret --value api_key=your_key_value --value domain=your_domain --value email=your_email
```
