# Anthropic

1. Go to [Anthropic Console](https://console.anthropic.com/) and create an API key.

2. Go to the **Settings** page in **Admyral** and click on **Add New Secret** in the **Secrets** section.

3. Give your secret a name, e.g. `Anthropic`.

4. The following secrets structure is expected:

| Key       | Value                                 |
| --------- | ------------------------------------- |
| `api_key` | Paste the API key from Anthropic here |

5. Click on the **Save** button.

Alternatively, you can use the following CLI command structure:

```bash
admyral secret set anthropic_secret --value api_key=your_key_value
```
