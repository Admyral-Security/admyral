# Abnormal

1. Go to [Anthropic Console](https://console.anthropic.com/) and create an API key.

2. Go to the **Settings** page in **Admyral** and click on **Add New Credential** in the **Credentials** section.

3. Give your credential a name, e.g. `Anthropic`.

4. The following secrets structure is expected:

| Key       | Value                                 |
| --------- | ------------------------------------- |
| `api_key` | Paste the API key from Anthropic here |

5. Click on the **Save** button.
