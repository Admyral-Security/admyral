# Assign Alert

Assign an alert in Opsgenie.

<Callout type="info">
	For Opsgenie's documentation, see [Assign
	Alert](https://docs.opsgenie.com/docs/alert-api#assign-alert).
</Callout>

## Parameters:

| Field               | Description                                                                                                                                                                                                                                                   | Required |
| ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :------: |
| **Identifier**      | Identifier of the alert.                                                                                                                                                                                                                                      |   Yes    |
| **Identifier Type** | Type of the identifier. Possible values are `id`, `tiny` and `alias`. Default value is `id`.                                                                                                                                                                  |    -     |
| **Owner**           | User that the alert will be assigned to. A JSON object with either `id` or `username` of the user should be provided. <br/> Example 1: `{"id": "4513b7ea-3b91-438f-b7e4-e3e54af9147c"}` <br/> Example 2: `{"username": ""username": "trinity@opsgenie.com""}` |   Yes    |
| **User**            | Display name of the request owner. Limit: 100 characters.                                                                                                                                                                                                     |    -     |
| **Source**          | Display name of the request source. Limit: 100 characters.                                                                                                                                                                                                    |    -     |
| **Note**            | Additional alert note to add. Limit: 25000 characters.                                                                                                                                                                                                        |    -     |
