## Add Tags to Alert

Add tags to an existing alert in Opsgenie.

<Callout type="info">
	For Opsgenie's documentation, see [Add Tags to
	Alert](https://docs.opsgenie.com/docs/alert-api#add-tags-to-alert).
</Callout>

### Parameters:

| Field               | Description                                                                                  | Required |
| ------------------- | -------------------------------------------------------------------------------------------- | :------: |
| **Identifier**      | Identifier of the alert.                                                                     |   Yes    |
| **Identifier Type** | Type of the identifier. Possible values are `id`, `tiny` and `alias`. Default value is `id`. |    -     |
| **Tags**            | A JSON array of tags to add to the alert. Example: `["tag1", "tag2"]`                        |   Yes    |
| **User**            | Display name of the request owner.                                                           |    -     |
| **Source**          | Display name of the request source.                                                          |    -     |
| **Note**            | Additional alert note to add.                                                                |    -     |
