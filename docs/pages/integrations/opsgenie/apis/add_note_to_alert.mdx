# Add Note to Alert

Add a note to an alert in Opsgenie.

<Callout type="info">
	For Opsgenie's documentation, see [Add Note to
	Alert](https://docs.opsgenie.com/docs/alert-api#add-note-to-alert).
</Callout>

## Parameters:

| Field               | Description                                                                                  | Required |
| ------------------- | -------------------------------------------------------------------------------------------- | :------: |
| **Identifier**      | Identifier of the alert.                                                                     |   Yes    |
| **Identifier Type** | Type of the identifier. Possible values are `id`, `tiny` and `alias`. Default value is `id`. |    -     |
| **User**            | Display name of the request owner. Limit: 100 characters.                                    |    -     |
| **Source**          | Display name of the request source. Limit: 100 characters.                                   |    -     |
| **Note**            | Alert note to add. Limit: 25000 characters.                                                  |   Yes    |
