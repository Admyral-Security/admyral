# List Alerts

List alerts in Opsgenie.

<Callout type="info">
	For Opsgenie's documentation, see [List
	Alerts](https://docs.opsgenie.com/docs/alert-api#list-alerts).
</Callout>

## Parameters:

| Field                      | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    | Required |
| -------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :------: |
| **Query**                  | Search query to apply while filtering the alerts. You can use field:value combination with most alert fields. See [here](https://support.atlassian.com/opsgenie/docs/search-queries-for-alerts/) for more information about search queries for alerts.                                                                                                                                                                                                                                                                                                         |    -     |
| **Search Identifier**      | Identifier of the saved search query to apply while filtering the alerts.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |    -     |
| **Search Identifier Type** | Identifier type of the saved search query. Possible values are `id` and `name`. Default value is `id`.                                                                                                                                                                                                                                                                                                                                                                                                                                                         |    -     |
| **Offset**                 | Start index of the result set (to apply pagination). Minimum value is `0`.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |    -     |
| **Limit**                  | Maximum number of items to provide in the result. Must be a positive integer value.                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |    -     |
| **Sort**                   | Name of the field that the result set will be sorted by. Possible values: <br/> - `createdAt` <br/> - `updatedAt` <br/> - `tinyId` <br/> - `alias` <br/> - `message` <br/> - `status` <br/> - `acknowledged` <br/> - `isSeen` <br/> - `snoozed` <br/> - `snoozedUntil` <br/> - `count` <br/> - `lastOccurredAt` <br/> - `source` <br/> - `owner` <br/> - `integration.name` <br/> - `integration.type` <br/> - `report.ackTime` <br/> - `report.closeTime` <br/> - `report.acknowledgedBy` <br/> - `report.closedBy` <br/> <br/> Default value is `createdAt`. |    -     |
| **Order**                  | Sorting order of the result set. Possible values are `desc` and `asc`. Default: `desc`                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |    -     |
