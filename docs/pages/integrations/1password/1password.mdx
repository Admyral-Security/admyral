# 1Password

1. In your 1Password console, go to the **Integrations** page and click **Other** below **Events Reporting**.

![1Password Integrations](/1password_integrations.png)

2. Create an integration by giving it a name (e.g., `Admyral`), then click **Add Integration**.

![1Password Integrations](/1password_token_creation.png)

3. Set up a token with your preferred expiration time and only activate **Audit Events**. Then, click **Issue Token**.

4. Copy the token.

5. Identify your API domain:

| If your account is on | Your API domain is:                                                                                    |
| --------------------- | ------------------------------------------------------------------------------------------------------ |
| `1Password.com`       | `events.1password.com` for 1Passwordd Business and `events.ent.1password.com` for 1Password Enterprise |
| `1Password.ca`        | `events.1password.ca`                                                                                  |
| `1Password.eu`        | `events.1password.eu`                                                                                  |

6. Go to the **Settings** page in **Admyral** and click on **Add New Secret** in the **Secrets** section.

7. Give your secret a name, e.g. `1Password`.

8. The following secrets structure is expected:

| Key       | Value                                 |
| --------- | ------------------------------------- |
| `api_key` | Paste the API key from 1Password here |
| `domain`  | Paste your API domain from above here |

9. Click on the **Save** button.

Alternatively, you can use the following CLI command structure:

```bash
admyral secret set 1password_secret --value api_key=your_key_value --value domain=your_api_domain
```
