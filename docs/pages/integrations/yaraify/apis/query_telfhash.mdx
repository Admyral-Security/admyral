# Query telfhash

You can get a list of recent files associated with a specific telfhash.

<Callout type="info">
	For YARAify's documentation, see [Query
	telfhash](https://yaraify.abuse.ch/api/#telfhash).
</Callout>

## Parameters:

| Field        | Description                | Required |
| ------------ | -------------------------- | :------: |
| **telfhash** | The telfhash to query for. |   Yes    |
