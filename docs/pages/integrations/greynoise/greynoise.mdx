# GreyNoise

1. In **GreyNoise**, click the Account icon in the top right corner. Next, click on **My API Key** in the dropdown.

2. Copy the API Key.

    <img src="/greynoise1.png" />

3. Go to the **Settings** page in **Admyral**. Click on **Create New Secret** in the **Secrets** section.

4. Give your credential a name, e.g. `GreyNoise`.

5. The following secrets structure is expected:

| Key       | Value                  |
| --------- | ---------------------- |
| `api_key` | Paste the API key here |

Alternatively, you can use the following CLI command structure:

```bash
admyral secret set greynoise_secret --value api_key=your_key_value
```

6. Click on the **Save** button.
