# GreyNoise

1. In **GreyNoise**, click the Account icon in the top right corner. Next, click on **My API Key** in the dropdown.

2. Copy the API Key.

    <img src="/greynoise1.png" />

3. Go to the **Settings** page in **Admyral**. Click on **Create New Credential** in the **Credentials** section. Select **GreyNoise** from the dropdown list. Give your credential a name, e.g. `GreyNoise`. Paste the token you copied in the previous step into the **API Key** field. Click **Save Credential**.
