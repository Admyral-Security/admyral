# IP Lookup

Lookup information about an IP address using GreyNoise.

<Callout type="info">
	For more information on the API for looking up an IP, see [GreyNoise IP
	Lookup via Community
	API](https://docs.greynoise.io/reference/get_v3-community-ip).
</Callout>

**SDK Import:**

```python
from admyral.actions import grey_noise_ip_lookup
```

## Arguments:

| Argument Name               | Description             | Required |
| --------------------------- | ----------------------- | :------: |
| **IP Address** `ip_address` | The IP address to query |   Yes    |

## Returns

A JSON object.

## Required Secrets

| Secret Placeholder  | Description                                                                |
| ------------------- | -------------------------------------------------------------------------- |
| `GREY_NOISE_SECRET` | GreyNoise secret. See [GreyNoise setup](/integrations/greynoise/greynoise) |

## SDK Example

```python
result = grey_noise_ip_lookup(
	ip_address="8.8.8.8",
	secrets={
		"GREY_NOISE_SECRET": "my_stored_greynoise_secret"
	}
)
```

## Example Output

```json
{
	"ip": "51.91.185.74",
```
