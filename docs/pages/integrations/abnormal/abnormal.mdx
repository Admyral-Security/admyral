# Abnormal

1. Follow the [Abnormal Documentation](https://abnormalsecurity.my.site.com/knowledgebase/s/article/Abnormal-REST-API-Integration#Integration-Steps) to get the API access token.

2. Go to the **Settings** page in **Admyral** and click on **Add New Credential** in the **Credentials** section.

3. Give your credential a name, e.g. `Abnormal`.

4. The following secrets structure is expected:

| Key       | Value                                |
| --------- | ------------------------------------ |
| `api_key` | Paste the API key from Abnormal here |

![Abnormal Secrets Setup](/abnormal_secrets.png)

Alternatively, you can use the following CLI command structure:

```bash
admyral secret set abnormal_secret --value api_key=your_key_value
```

5. Click on the **Save** button.
