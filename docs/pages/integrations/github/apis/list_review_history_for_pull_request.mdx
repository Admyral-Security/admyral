# List Review History for a Pull Request

This method enables users to list the review history for a pull request in a GitHub repository.

<Callout type="info">
	For more information on the GitHub API endpoint for pull request reviews,
	see
	[Here](https://docs.github.com/en/rest/pulls/reviews?apiVersion=2022-11-28).
</Callout>

To use this API endpoint, your GitHub token requires the following scopes:

```
- "Pull requests" repository permissions (read)
```

**SDK Import:**

```python
from admyral.actions import list_review_history_for_pull_request
```

## Arguments

| Argument Name                                 | Description                                                                           | Required |
| --------------------------------------------- | ------------------------------------------------------------------------------------- | :------: |
| **Repository Owner** `repo_owner`             | The owner of the repository.                                                          |   yes    |
| **Repository Name** `filter`                  | The name of the repository.                                                           |   yes    |
| **Pull Request Number** `pull_request_number` | The unique number of the pull request.                                                |   yes    |
| **State** `state`                             | The review state, can be one of \{APPROVED, CHANGES_REQUESTED, COMMENTED, DISMISSED}. |    -     |

## Returns

A JSON array of commits.

<Callout type="info">
	For more information on the API response for listing reviews on a pull
	requests, see [List commits on a pull
	request](https://docs.github.com/en/rest/pulls/reviews?apiVersion=2022-11-28#list-reviews-for-a-pull-request).
</Callout>

## Required Secrets

| Secret Placeholder | Description                                                    |
| ------------------ | -------------------------------------------------------------- |
| `GITHUB_SECRET`    | GitHub secret. See [GitHub setup](/integrations/github/github) |

## SDK Example

```python
commit_diff = list_review_history_for_pull_request(
	repo_owner="admyral",
	repo_name="admyral",
	pull_request_number="3",
	state="APPROVED",
	secrets={"GITHUB_SECRET": "your_github_secret"}
)
```
