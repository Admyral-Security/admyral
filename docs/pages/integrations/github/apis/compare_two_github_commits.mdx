# Compare Two Commits

This method enables users to compare two commits in a GitHub repository. Events include who authored and committed the changes, and the changes made.

<Callout type="info">
	For more information on the GitHub API endpoint for commits, see
	[Here](https://docs.github.com/en/rest/commits/commits?apiVersion=2022-11-28).
</Callout>

To use this API endpoint, your GitHub token requires the following scopes:

```
- "Contents" repository permissions (read)
```

**SDK Import:**

```python
from admyral.actions import compare_two_github_commits
```

## Arguments

| Argument Name                     | Description                                              | Required |
| --------------------------------- | -------------------------------------------------------- | :------: |
| **Repository Owner** `repo_owner` | The owner of the repository.                             |   yes    |
| **Repository Name** `filter`      | The name of the repository.                              |   yes    |
| **Base** `base`                   | The base commit.                                         |   yes    |
| **Head** `head`                   | The head commit.                                         |   yes    |
| **Diff Type** `diff_type`         | The type of diff to return, can be one of \{json, diff}. |    -     |

## Returns

A JSON of the commit diff.

<Callout type="info">
	For more information on the API response for comparing commits, see [Compare
	two
	commits](https://docs.github.com/en/rest/commits/commits?apiVersion=2022-11-28#compare-two-commits).
</Callout>

## Required Secrets

| Secret Placeholder | Description                                                    |
| ------------------ | -------------------------------------------------------------- |
| `GITHUB_SECRET`    | GitHub secret. See [GitHub setup](/integrations/github/github) |

## SDK Example

```python
commit_diff = compare_two_github_commits(
	repo_owner="admyral",
	repo_name="admyral",
	base="your_base_commit",
	head="your_head_commit",
	secrets={"GITHUB_SECRET": "your_github_secret"}
)
```
