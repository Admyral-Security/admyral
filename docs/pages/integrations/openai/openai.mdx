# OpenAI

1. Visit the [OpenAI Platform](https://platform.openai.com) and log in to your account.

2. In the top-left, click on your project and then on **Organization overview**.

3. Click on **API keys** in the left sidebar.

![OpenAI API Keys](/openai_api_keys.png)

4. Click on **+ Create new secret key** and create a secret key.

5. Copy the secret key.

6. Go to the **Settings** page in **Admyral**. Click on **Create New Secret** in the **Secrets** section.

7. Give your credential a name, e.g. `OpenAI`.

8. The following secrets structure is expected:

| Key       | Value                       |
| --------- | --------------------------- |
| `api_key` | Paste your **API key** here |

Alternatively, you can use the following CLI command structure:

```bash
admyral secret set openai_secret --value api_key=your_key_value
```

9. Click on the **Save** button.
