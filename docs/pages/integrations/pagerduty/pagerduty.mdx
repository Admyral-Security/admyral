# PagerDuty

1. Go to your [PagerDuty console](https://pagerduty.com) and log in.

2. Click on **Integrations** in the top navigation bar and then on **API Access Keys**.

![PagerDuty](/pagerduty_connect.png)

3. Click on **+ Create New API Key**. Create the API key with write access.

4. Copy the API key.

5. Go to the **Settings** page in **Admyral**. Click on **Create New Secret** in the **Secrets** section.

6. Give your credential a name, e.g. `PagerDuty`.

7. The following secrets structure is expected:

| Key       | Value                          |
| --------- | ------------------------------ |
| `api_key` | Paste your **API key** here    |
| `email`   | Paste your **user email** here |

8. Click on the **Save** button.
