# VirusTotal

1. In **VirusTotal**, click the Settings icon in the top right corner. Next, click on **API Key**.

2. Copy the API Key.

<img src="/virustotal1.png" />

3. Go to the **Settings** page in **Admyral**. Click on **Create New Secret** in the **Secrets** section.

4. Give your credential a name, e.g. `VirusTotal`.

5. The following secrets structure is expected:

| Key       | Value                       |
| --------- | --------------------------- |
| `api_key` | Paste your **API Key** here |

Alternatively, you can use the following CLI command structure:

```bash
admyral secret set virus_total_secret --value api_key=your_key_value
```

6. Click on the **Save** button.
