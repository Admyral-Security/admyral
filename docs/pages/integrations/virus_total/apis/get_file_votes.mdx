# Get File Votes

Retrieve the votes for a file.

<Callout type="info">
	For VirusTotal's documentation, see [Get File
	Votes](https://docs.virustotal.com/reference/files-votes-get).
</Callout>

## Parameters:

| Field       | Description                                                                 | Required |
| ----------- | --------------------------------------------------------------------------- | :------: |
| **File ID** | The unique identifier (hash) of the file for which votes will be retrieved. |   Yes    |
