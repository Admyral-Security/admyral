# Get Domain Comments

Retrieve the comments for a domain.

<Callout type="info">
	For VirusTotal's documentation, see [Get Domain
	Comments](https://docs.virustotal.com/reference/domains-comments-get).
</Callout>

## Parameters:

| Field      | Description                                      | Required |
| ---------- | ------------------------------------------------ | :------: |
| **Domain** | The domain for which comments will be retrieved. |   Yes    |
