# Abnormal

1. Go to [AlienVault OTX API Documentation](https://otx.alienvault.com/api). You can find your OTX API key in the top-right.

2. Go to the **Settings** page in **Admyral** and click on **Add New Credential** in the **Credentials** section.

3. Give your credential a name, e.g. `AlienVault`.

4. The following secrets structure is expected:

| Key       | Value                                      |
| --------- | ------------------------------------------ |
| `api_key` | Paste the API key from AlienVault OTX here |

5. Click on the **Save** button.
