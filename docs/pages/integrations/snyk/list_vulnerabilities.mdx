# List Vulnerabilities (Snyk)

Retrieve a list of vulnerabilities from Snyk within a specified time range. This API allows filtering based on organization ID, start and end times, and setting a limit on the number of results.

<Callout type="info">
	For more information on the Snyk API, see [Snyk API
	Documentation](https://apidocs.snyk.io/?version=2024-06-21#get-/orgs/-org_id-/issues).
</Callout>

## Parameters:

| Field               | Description                                                           | Required |
| ------------------- | --------------------------------------------------------------------- | :------: |
| **Organization ID** | The ID of the organization to list vulnerabilities for.               |   Yes    |
| **Start Time**      | The start time of the vulnerabilities in ISO-8601 format (inclusive). |    No    |
| **End Time**        | The end time of the vulnerabilities in ISO-8601 format (inclusive).   |    No    |
| **Limit**           | The maximum number of vulnerabilities to return. The default is 1000. |    No    |
