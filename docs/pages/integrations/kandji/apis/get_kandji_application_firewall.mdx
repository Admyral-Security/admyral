# Application Firewall

Get Application Firewall details for macOS.

<Callout type="info">
	For more information on the API, see [Application
	Firewall](https://api-docs.kandji.io/#0d3abc6c-a5a5-4fe6-b7d4-19b1287eaf91).
</Callout>

**Required permissions:**

-   `Application Firewall`

**SDK Import:**

```python
from admyral.actions import get_kandji_application_firewall
```

## Arguments:

| Argument Name | Description | Required |
| ------------- | ----------- | :------: |

## Returns

A list of JSON objects.

## Required Secrets

| Secret Placeholder | Description                                                    |
| ------------------ | -------------------------------------------------------------- |
| `KANDJI_SECRET`    | Kandji secret. See [Kandji setup](/integrations/kandji/kandji) |

## SDK Example

```python
apps = get_kandji_application_firewall(
	secrets={"KANDJI_SECRET": "my_kandji_secret"}
)
```
