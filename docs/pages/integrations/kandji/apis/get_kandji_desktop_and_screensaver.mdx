# Desktop & Screensaver

Get Desktop and Screensaver details for macOS.

<Callout type="info">
	For more information on the API, see [Desktop and
	Screensaver](https://api-docs.kandji.io/#7f6f6813-6522-4249-9f6a-3d0c479e2bbe).
</Callout>

**Required permissions:**

-   `Desktop & Screensaver`

**SDK Import:**

```python
from admyral.actions import get_kandji_desktop_and_screensaver
```

## Arguments:

| Argument Name | Description | Required |
| ------------- | ----------- | :------: |

## Returns

A list of JSON objects.

## Required Secrets

| Secret Placeholder | Description                                                    |
| ------------------ | -------------------------------------------------------------- |
| `KANDJI_SECRET`    | Kandji secret. See [Kandji setup](/integrations/kandji/kandji) |

## SDK Example

```python
apps = get_kandji_desktop_and_screensaver(
	secrets={"KANDJI_SECRET": "my_kandji_secret"}
)
```
