# Get Device Details

Get details for a Kandji managed device.

<Callout type="info">
	For more information on the API, see [Get Device
	Details](https://api-docs.kandji.io/#efa2170d-e5f7-4b97-8f4c-da6f84ba58b5).
</Callout>

**Required permissions:**

-   `Device details`

**SDK Import:**

```python
from admyral.actions import get_kandji_device_details
```

## Arguments:

| Argument Name             | Description           | Required |
| ------------------------- | --------------------- | :------: |
| **Device ID** `device_id` | The ID of the device. |   Yes    |

## Returns

A JSON object.

## Required Secrets

| Secret Placeholder | Description                                                    |
| ------------------ | -------------------------------------------------------------- |
| `KANDJI_SECRET`    | Kandji secret. See [Kandji setup](/integrations/kandji/kandji) |

## SDK Example

```python
device_details = get_kandji_device_details(
    device_id="03f81208-2b6a-4a77-81f5-cf1633bcfb95",
	secrets={"KANDJI_SECRET": "my_kandji_secret"}
)
```
