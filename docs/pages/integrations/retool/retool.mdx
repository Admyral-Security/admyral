# Retool

1. Go to [Retool](https://retool.com) and log in.

2. Go to your **Settings** and in the left sidebar click on **Retool API**.

3. Click on **Create new** and create one.

![Retool API](/retool_api.png)

4. Copy your API key.

5. Go to the **Settings** page in **Admyral**. Click on **Create New Secret** in the **Secrets** section.

6. Give your credential a name, e.g. `Retool`.

7. The following secrets structure is expected:

| Key       | Value                             |
| --------- | --------------------------------- |
| `api_key` | Paste your **API key** here       |
| `domain`  | Paste your **Retool domain** here |

Alternatively, you can use the following CLI command structure:

```bash
admyral secret set retool_secret --value api_key=your_key_value --value domain=your_domain
```

8. Click on the **Save** button.
