# Explore

Retrieve information from Pulsedive using the Explore API.

<Callout type="info">
  For Pulsedive's documentation, see [Explore API](https://pulsedive.com/api/explore).
</Callout>

## Parameters:

| Field          | Description                                                          | Required |
| -------------- | -------------------------------------------------------------------- | :------: |
| **Query**      | The query to search Pulsedive using the Explore query language.      |   Yes    |
| **Limit**      | The number of records to return.                                     |    No    |
