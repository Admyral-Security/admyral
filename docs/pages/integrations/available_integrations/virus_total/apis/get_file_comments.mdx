# Get File Comments

Retrieve the comments for a file.

<Callout type="info">
	For VirusTotal's documentation, see [Get File Comments](https://docs.virustotal.com/reference/files-comments-get).
</Callout>

## Parameters:

| Field       | Description                                                                                 | Required |
| ----------- | ------------------------------------------------------------------------------------------- | :------: |
| **File ID** | The unique identifier (hash) of the file for which comments will be retrieved.              |   Yes    |
