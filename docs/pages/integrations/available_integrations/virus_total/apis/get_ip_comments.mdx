# Get IP Comments

Retrieve the comments for an IP address.

<Callout type="info">
	For VirusTotal's documentation, see [Get IP Comments](https://docs.virustotal.com/reference/ip-comments-get).
</Callout>

## Parameters:

| Field       | Description                                                                                 | Required |
| ----------- | ------------------------------------------------------------------------------------------- | :------: |
| **IP**      | The IP address for which comments will be retrieved.                                        |   Yes    |
