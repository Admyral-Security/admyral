# Search API

Search for files, URLs, domains, or IP addresses.

<Callout type="info">
	For VirusTotal's documentation, see [Search API](https://docs.virustotal.com/reference/api-search).
</Callout>

## Parameters:

| Field       | Description                                                                                 | Required |
| ----------- | ------------------------------------------------------------------------------------------- | :------: |
| **Query**   | The search query (e.g., file hash, domain, IP address, URL).                                |   Yes    |
