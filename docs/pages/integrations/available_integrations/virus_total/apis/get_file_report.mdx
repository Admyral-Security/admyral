# Retrieve File Information

Retrieve detailed information about a file from VirusTotal.

<Callout type="info">
	For VirusTotal's documentation, see [Retrieve File Information](https://docs.virustotal.com/reference/file-info).
</Callout>

## Parameters:

| Field            | Description                                                                                                                                                                                                                                                                                          | Required |
| ---------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :------: |
| **File ID**      | The unique identifier (SHA-256 hash) of the file to retrieve information for.                                                                                                                                                                                                                        |   Yes    |
