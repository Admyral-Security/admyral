# Get URL Votes

Retrieve votes for a specific URL.

<Callout type="info">
	For VirusTotal's documentation, see [Get URL
	Votes](https://docs.virustotal.com/reference/urls-votes-get).
</Callout>

## Parameters:

| Field      | Description                                                                                                                                      | Required |
| ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------ | :------: |
| **URL ID** | The encoded URL you want to retrieve votes for. Note that the URL should be encoded in base64 format (e.g., `aHR0cDovL3d3dy5leGFtcGxlLmNvbQ==`). |   Yes    |
