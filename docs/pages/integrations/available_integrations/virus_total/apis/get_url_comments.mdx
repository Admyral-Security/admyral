# Get URL Comments

Retrieve the comments for a URL.

<Callout type="info">
	For VirusTotal's documentation, see [Get URL Comments](https://docs.virustotal.com/reference/urls-comments-get).
</Callout>

## Parameters:

| Field       | Description                                                                                 | Required |
| ----------- | ------------------------------------------------------------------------------------------- | :------: |
| **URL ID**  | The unique identifier (hash) of the URL for which comments will be retrieved.               |   Yes    |
