# Get IP Votes

Retrieve votes for a specific IP address.

<Callout type="info">
	For VirusTotal's documentation, see [Get IP Votes](https://docs.virustotal.com/reference/ip-votes).
</Callout>

## Parameters:

| Field             | Description                                                                                                                                                                                | Required |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | :------: |
| **IP Address**    | The IP address you want to retrieve votes for.                                                                                                                                              |   Yes    |
