# URL Info

Retrieve detailed information about a URL.

<Callout type="info">
	For VirusTotal's documentation, see
	[URL Info](https://docs.virustotal.com/reference/url-info).
</Callout>

## Parameters:

| Field               | Description                                                                                                       | Required |
| ------------------- | ----------------------------------------------------------------------------------------------------------------- | :------: |
| **URL**             | The URL to retrieve information for.                                                                              |   Yes    |
