# Get an IP Address Report

Retrieve detailed information about a specific IP address.

<Callout type="info">
	For VirusTotal's documentation, see [Get IP Address Report](https://developers.virustotal.com/reference/ip-addresses).
</Callout>

## Parameters:

| Field       | Description                                                                       | Required |
| ----------- | --------------------------------------------------------------------------------- | :------: |
| **IP**      | The IP address to retrieve the report for.                                        |   Yes    |
