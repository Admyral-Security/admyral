# Get Domain Votes

Retrieve the votes for a domain.

<Callout type="info">
	For VirusTotal's documentation, see [Get Domain
	Votes](https://docs.virustotal.com/reference/domains-votes-get).
</Callout>

## Parameters:

| Field      | Description                                   | Required |
| ---------- | --------------------------------------------- | :------: |
| **Domain** | The domain for which votes will be retrieved. |   Yes    |
