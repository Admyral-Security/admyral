# List Users

Lists all users in a Slack team.

<Callout type="info">
	For Slack's documentation, see
	[users.list](https://api.slack.com/methods/users.list).
</Callout>

## Parameters:

| Field                         | Description                                                                                  | Required |
| ----------------------------- | -------------------------------------------------------------------------------------------- | :------: |
| **Include Locale for Users?** | Set this to true to receive the locale for users. Defaults to false.                         |    -     |
| **Limit**                     | Number of users to return per page. Maximum of 1000. Defaults to 200.                        |    -     |
| **Return All Pages**          | Set this to true to handle pagination automatically and return all users. Defaults to false. |    -     |
