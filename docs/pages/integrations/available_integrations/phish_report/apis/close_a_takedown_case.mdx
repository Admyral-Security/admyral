# Close Takedown Case

Close a case (e.g. because you've confirmed the site is no longer active).

<Callout type="info">
	For Phish Reports's documentation, see [Start a Takedown
	API](https://phish.report/api/v0#tag/Takedown/paths/~1api~1v0~1cases/post).
</Callout>

## Parameters:

| Field                 | Description                                                     | Required |
| --------------------- | --------------------------------------------------------------- | :------: |
| **Case ID**           | The ID of the takedown request.                                 |   Yes    |
| **Comment**           | An optional comment explaining why the case is being closed.    |    No    |

## Example Output:
```json
"string"
```
