# List channels

Retrieve the list of channels in a Microsoft Teams team.

<Callout type="info">
    For Microsoft's documentation, see [List channels](https://learn.microsoft.com/en-us/graph/api/channel-list?view=graph-rest-1.0&tabs=http#http-request).
</Callout>

## Parameters:

| Field        | Description                                                          | Required |
| ------------ | -------------------------------------------------------------------- | :------: |
| **Team ID**  | The ID of the team to retrieve channels from.                        |   Yes    |

## Example Output:

```json
{
  "value": [
    {
      "id": "19:561fbdbbfca848a484f0a6f00ce9dbbd@thread.tacv2",
      "createdDateTime": "2020-05-27T19:22:25.692Z",
      "displayName": "General",

      "membershipType": "standard"
    }
  ]
}
