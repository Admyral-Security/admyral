# Create Chat

Creates a new chat in Microsoft Teams.

> Note: Only one one-on-one chat can exist between two members. If a one-on-one chat already exists, this operation will return the existing chat and not create a new one.

<Callout type="info">
	For Microsoft Teams' documentation, see [Create
	Chat](https://learn.microsoft.com/en-us/graph/api/chat-post?view=graph-rest-1.0&tabs=http#http-request).
</Callout>

## Parameters:

| Field        | Description                                                                                                                                                                                                                                  | Required |
| ------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :------: |
| **members**  | The members to add to the chat. This should be an array of member objects, where each object contains the `@odata.type` (which should be `#microsoft.graph.aadUserConversationMember`), `roles`, and `user@odata.bind` (with the user's ID). |   Yes    |
| **chatType** | The type of the chat. Can be `group` or `oneOnOne`.                                                                                                                                                                                          |   Yes    |
| **topic**    | The topic of the chat.                                                                                                                                                                                                                       |    No    |

Example format for members:

```json
[
	{
		"@odata.type": "#microsoft.graph.aadUserConversationMember",
		"roles": ["owner"],
		"user@odata.bind": "https://graph.microsoft.com/v1.0/users('<user_id_of_first_member>')"
	},
	{
		"@odata.type": "#microsoft.graph.aadUserConversationMember",
		"roles": ["owner"],
		"user@odata.bind": "https://graph.microsoft.com/v1.0/users('<user_id_of_second_member>')"
	}
]
```

## Example Output:

