# Abnormal

1. Follow the [Abnormal Documentation](https://abnormalsecurity.my.site.com/knowledgebase/s/article/Abnormal-REST-API-Integration#Integration-Steps) to get the API access token.

2. Go to the **Settings** page in **Admyral**. Click on **Create New Credential** in the **Credentials** section. Select **Abnormal** from the dropdown list. Give your credential a name, e.g. `Abnormal`. Paste the token you copied in the previous step into the **API Key** field. Click **Save Credential**.
