# Get Case Analysis

Provides the analysis and timeline details of a case.

<Callout type="info">
	For Abnormal's documentation, see [Abnormal Get Case Analysis
	API](https://app.swaggerhub.com/apis-docs/abnormal-security/abx/1.4.1#/Cases/get_cases__caseId__analysis).
</Callout>

## Parameters:

| Field       | Description          | Required |
| ----------- | -------------------- | :------: |
| **Case ID** | The Case ID to query |   Yes    |
