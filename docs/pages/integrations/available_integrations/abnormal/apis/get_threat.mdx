# Get Threat

Get details of a threat.

<Callout type="info">
	For Abnormal's documentation, see [Abnormal Get Threat
	API](https://app.swaggerhub.com/apis-docs/abnormal-security/abx/1.4.3#/Threats/get_threats__threatId_).
</Callout>

## Parameters:

| Field         | Description                    | Required |
| ------------- | ------------------------------ | :------: |
| **Threat ID** | A UUID representing the threat |   Yes    |
