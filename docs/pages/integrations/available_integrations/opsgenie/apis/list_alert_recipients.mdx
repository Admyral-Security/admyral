## List Alert Recipients

List the recipients of an existing alert in Opsgenie.

<Callout type="info">
	For Opsgenie's documentation, see [List Alert
	Recipients](https://docs.opsgenie.com/docs/alert-api#list-alert-recipients).
</Callout>

### Parameters:

| Field               | Description                                                                                                                           | Required |
| ------------------- | ------------------------------------------------------------------------------------------------------------------------------------- | :------: |
| **Identifier**      | Identifier of the alert.                                                                                                              |   Yes    |
| **Identifier Type** | Type of the identifier that is provided as an in-line parameter. Possible values are `id`, `tiny` and `alias`. Default value is `id`. |    -     |
