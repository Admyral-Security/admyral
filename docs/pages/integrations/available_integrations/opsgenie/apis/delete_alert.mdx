# Delete Alert

Delete an alert in Opsgenie based on id, tiny id, or alias.

<Callout type="info">
	For Opsgenie's documentation, see [Delete
	Alert](https://docs.opsgenie.com/docs/alert-api#delete-alert).
</Callout>

## Parameters:

| Field               | Description                                                                                       | Required |
| ------------------- | ------------------------------------------------------------------------------------------------- | :------: |
| **Identifier**      | Identifier of the alert.                                                                          |   Yes    |
| **Identifier Type** | Type of the identifier that is provided. Possible values: `id`, `tiny`, or `alias`. Default: `id` |    -     |
| **User**            | Display name of the request owner. Limit: 100 characters.                                         |    -     |
| **Source**          | Display name of the request source. Limit: 100 characters.                                        |    -     |
