## Add Details (Custom Properties) to Alert

Add custom properties (details) to an existing alert in Opsgenie.

<Callout type="info">
	For Opsgenie's documentation, see [Add Details to
	Alert](https://docs.opsgenie.com/docs/alert-api#add-details-custom-properties-to-alert).
</Callout>

### Parameters:

| Field               | Description                                                                                                                         | Required |
| ------------------- | ----------------------------------------------------------------------------------------------------------------------------------- | :------: |
| **Identifier**      | Identifier of the alert.                                                                                                            |   Yes    |
| **Identifier Type** | Type of the identifier. Possible values are `id`, `tiny` and `alias`. Default value is `id`.                                        |    -     |
| **Details**         | Key-value pairs represented as a JSON object to add as custom properties to the alert. Example: `{"key1":"value1","key2":"value2"}` |   Yes    |
| **User**            | Display name of the request owner.                                                                                                  |    -     |
| **Source**          | Display name of the request source.                                                                                                 |    -     |
| **Note**            | Additional alert note to add.                                                                                                       |    -     |
