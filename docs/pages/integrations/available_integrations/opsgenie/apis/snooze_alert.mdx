# Snooze Alert

Snooze an alert in Opsgenie.

<Callout type="info">
    For Opsgenie's documentation, see [Snooze Alert](https://docs.opsgenie.com/docs/alert-api#snooze-alert).
</Callout>

## Parameters:

| Field            | Description                                                                                             | Required |
| ---------------- | ------------------------------------------------------------------------------------------------------- | :------: |
| **Identifier**   | Identifier of the alert.                                                                                |   Yes    |
| **IdentifierType** | Type of the identifier that is provided as an in-line parameter. Possible values are id, tiny and alias. Default value is id. |    -     |
| **End Time**     | Date and time that snooze will lose effect. Provided value should be in ISO 8601 format.                 |   Yes    |
| **User**         | Display name of the request owner. Limit: 100 characters.                                               |    -     |
| **Source**       | Display name of the request source. Limit: 100 characters.                                              |    -     |
| **Note**         | Additional alert note to add. Limit: 25000 characters.                                                  |    -     |

## Example Request:


<Callout type="info">
    Acknowledge alert requests are processed asynchronously, therefore valid requests are responded to with `HTTP status 202 - Accepted`.
</Callout>
