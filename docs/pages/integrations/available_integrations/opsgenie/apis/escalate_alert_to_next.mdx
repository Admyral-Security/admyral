# Escalate Alert to Next

Escalate an alert to the next step in Opsgenie.

<Callout type="info">
	For Opsgenie's documentation, see [Escalate Alert to
	Next](https://docs.opsgenie.com/docs/alert-api#escalate-alert-to-next).
</Callout>

## Parameters:

| Field               | Description                                                                                                                                                                                                                                      | Required |
| ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | :------: |
| **Identifier**      | Identifier of the alert.                                                                                                                                                                                                                         |   Yes    |
| **Identifier Type** | Type of the identifier. Possible values are `id`, `tiny` and `alias`. Default value is `id`.                                                                                                                                                     |    -     |
| **Escalation**      | Escalation that the alert will be escalated. A JSON object with either `id` or `name` of the escalation should be provided. <br/> Example 1: `{"id":"4513b7ea-3b91-438f-b7e4-e3e54af9147c"}` <br/> Example 2: `{"name":"Operations_Escalation"}` |   Yes    |
| **User**            | Display name of the request owner. Limit: 100 characters.                                                                                                                                                                                        |    -     |
| **Source**          | Display name of the request source. Limit: 100 characters.                                                                                                                                                                                       |    -     |
| **Note**            | Additional alert note to add. Limit: 25000 characters.                                                                                                                                                                                           |    -     |
