## Add Responder to Alert

Add a responder to an existing alert in Opsgenie.

<Callout type="info">
	For Opsgenie's documentation, see [Add Responder to
	Alert](https://docs.opsgenie.com/docs/alert-api#add-responder-to-alert).
</Callout>

### Parameters:

| Field               | Description                                                                                                                                                                                                                                               | Required |
| ------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :------: |
| **Identifier**      | Identifier of the alert.                                                                                                                                                                                                                                  |   Yes    |
| **Identifier Type** | Type of the identifier. Possible values are `id`, `tiny` and `alias`. Default value is `id`.                                                                                                                                                              |    -     |
| **Responder**       | Team or user that the alert will be routed to. A JSON object with either type `team` or `user`. For type `team`, either the `id` or `name` of the team should be provided. For type `user`, either the `id` or `username` of the user should be provided. |   Yes    |
| **User**            | Display name of the request owner.                                                                                                                                                                                                                        |    -     |
| **Source**          | Display name of the request source.                                                                                                                                                                                                                       |    -     |
| **Note**            | Additional alert note to add.                                                                                                                                                                                                                             |    -     |

**Responder** examples:

```json
{
	"responder": {
		"type": "team",
		"id": "4513b7ea-3b91-438f-b7e4-e3e54af9147c"
	}
}
```

```json
{
	"responder": {
		"type": "team",
		"name": "secops"
	}
}
```

```json
{
	"responder": {
		"username": "trinity@opsgenie.com",
		"type": "user"
	}
}
```

```json
{
	"responder": {
		"id": "4513b7ea-3b91-438f-b7e4-e3e54af9147c",
		"type": "user"
	}
}
```
