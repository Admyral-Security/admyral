## Update Alert Message

Update the message of an existing alert in Opsgenie.

<Callout type="info">
	For Opsgenie's documentation, see [Update Alert
	Message](https://docs.opsgenie.com/docs/alert-api#update-alert-message).
</Callout>

### Parameters:

| Field               | Description                                                                                                                           | Required |
| ------------------- | ------------------------------------------------------------------------------------------------------------------------------------- | :------: |
| **Identifier**      | Identifier of the alert.                                                                                                              |   Yes    |
| **Identifier Type** | Type of the identifier that is provided as an in-line parameter. Possible values are `id`, `tiny` and `alias`. Default value is `id`. |    -     |
| **Message**         | Message of the alert.                                                                                                                 |   Yes    |
