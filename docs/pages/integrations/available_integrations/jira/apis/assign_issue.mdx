# Assign Issue

Assigns an issue to a user.

<Callout type="info">
	For Jira's documentation, see [Assign
	Issue](https://developer.atlassian.com/cloud/jira/platform/rest/v3/api-group-issues/#api-rest-api-3-issue-issueidorkey-assignee-put).
</Callout>

## Parameters:

| Field               | Description                                                                                                                                                                                              | Required |
| ------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :------: |
| **Issue ID or Key** | The ID or key of the issue to be assigned.                                                                                                                                                               |   Yes    |
| **Account ID**      | The account ID of the assignee. You can obtain the ID in the UI (see [here](https://community.atlassian.com/t5/Jira-questions/where-can-i-find-my-Account-ID/qaq-p/976527)) or use the `Find users` API. |   Yes    |
