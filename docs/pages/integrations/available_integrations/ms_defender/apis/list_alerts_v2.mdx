# List Alerts V2

Get a list of alert resources created to track suspicious activities in an organization.

<Callout type="info">
	For Microsoft Security Graph API's documentation, see [List Alerts
	v2](https://learn.microsoft.com/en-us/graph/api/security-list-alerts_v2?view=graph-rest-1.0&tabs=http).
</Callout>

## Parameters:

| Field            | Description                                                                                                                                                                                                                                                                                                     | Required |
| ---------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :------: |
| **Filter**       | An OData filter to apply to the alert list. The following properties support filter: assignedTo, classification, determination, createdDateTime, lastUpdateDateTime, severity, serviceSource and status. Example: `serviceSource eq 'microsoftDefenderForEndpoint' and createdDateTime gt 2021-01-01T00:00:00Z` |    -     |
| **Limit**        | The maximum number of alerts to return.                                                                                                                                                                                                                                                                         |    -     |
| **Skip**         | The number of alerts to skip before returning the results.                                                                                                                                                                                                                                                      |    -     |
| **Count Alerts** | Instead of returning all the results, the count parameter returns the number of items in the result set. Default: false                                                                                                                                                                                         |    -     |

### Filter Query

The following operators are supported for the `filter` query parameter:

| Operator Type             | Operator                                                                                                                    |
| ------------------------- | --------------------------------------------------------------------------------------------------------------------------- |
| **Equality operators**    | - Equals (`eq`) <br/> - Not Equals (`ne`) <br/> - Logical negation (`not`) <br/> - In (`in`) <br/> - Has (`has`)            |
| **Relational operators**  | - Less than (`lt`) <br/> - Greater than (`gt`) <br/> - Less than or equal to (`le`) <br/> - Greater than or equal to (`ge`) |
| **Lambda operators**      | - Any (`any`) <br/> - All (`all`)                                                                                           |
| **Conditional operators** | - And (`and`) <br/> - Or (`or`)                                                                                             |
| **Functions**             | - Starts with (`startswith`) <br/> - Ends with (`endswith`) <br/> - Contains (`contains`)                                   |

The API supports the following properties for the `filter` query parameter:

-   `assignedTo`
-   `classification`
-   `determination`
-   `createdDateTime`
-   `lastUpdateDateTime`
-   `severity`
-   `serviceSource`
-   `status`

See [here](https://learn.microsoft.com/en-us/graph/filter-query-parameter?tabs=http) for more information on filter queries.

### Example Filter Queries

-   `createdDateTime gt 2024-05-01T00:00:00Z`
-   `serviceSource eq 'microsoftDefenderForEndpoint' and createdDateTime gt 2021-01-01T00:00:00Z`
