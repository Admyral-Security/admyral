# Create Comment for Incident 

Create a comment for an incident.

<Callout type="info">
	For Microsoft's documentation, see [Microsoft Defender Create Comment for Incident API](https://learn.microsoft.com/en-us/graph/api/security-incident-post-comments?view=graph-rest-1.0&tabs=http).
</Callout>

## Parameters

| Field            | Description                                               | Required |
| ---------------- | --------------------------------------------------------- | :------: |
| **Incident ID**  | The ID of the incident to comment on.	                   |   Yes    |
| **Comment**      | The comment to add to the alert.	                       |   Yes    |
