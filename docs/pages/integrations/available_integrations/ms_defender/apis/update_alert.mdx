# Update Alert

Update the status, classification, determination, and assignedTo fields of a specific alert.

<Callout type="info">
  For Microsoft's documentation, see [Microsoft Defender Update Alert API](https://learn.microsoft.com/en-us/graph/api/security-alert-update?view=graph-rest-1.0&tabs=http).
</Callout>

## Parameters:

| Field            | Description                                                                                           | Required |
|------------------|-------------------------------------------------------------------------------------------------------|:--------:|
| **Alert ID**     | The ID of the alert to update.                                                                        |   Yes    |
| **Status**       | The status of the alert. Possible values: new, inProgress, resolved, unknownFutureValue.              |   No     |
| **Classification** | The classification of the alert. Possible values: unknown, falsePositive, truePositive, informationalExpectedActivity, unknownFutureValue. |   No     |
| **Determination** | The determination of the alert. Possible values: unknown, apt, malware, securityPersonnel, securityTesting, unwantedSoftware, other, multiStagedAttack, compromisedUser, phishing, maliciousUserActivity, clean, insufficientData, confirmedUserActivity, lineOfBusinessApplication, unknownFutureValue. |   No     |
| **Assigned To**  | Owner of the incident, or null if no owner is assigned.                                               |   No     |
