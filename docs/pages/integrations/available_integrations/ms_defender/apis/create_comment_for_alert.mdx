# Create Comment for Alert

Create a comment for an alert.

<Callout type="info">
	For Microsoft's documentation, see [Microsoft Defender Create Comment for Alert API](https://learn.microsoft.com/en-us/graph/api/security-alert-post-comments?view=graph-rest-1.0&tabs=http).
</Callout>

## Parameters

| Field           | Description                                               | Required |
| --------------- | --------------------------------------------------------- | :------: |
| **Alert ID**    | The ID of the alert to comment on.	                      |   Yes    |
| **Comment**     | The comment to add to the alert.	                      |    Yes    |
