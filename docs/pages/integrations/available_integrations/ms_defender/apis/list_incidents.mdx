# List Incidents

Get a list of incident resources created to track attacks in an organization.

<Callout type="info">
    For Microsoft's documentation, see [Microsoft Defender List Incidents API](https://learn.microsoft.com/en-us/graph/api/security-list-incidents?view=graph-rest-1.0&tabs=http).
</Callout>

## Parameters

| Field           | Description                                                                                                      | Required |
| --------------- | ---------------------------------------------------------------------------------------------------------------- | :------: |
| **OData Filter** | An OData filter to apply to the incident list. The following properties support filter: assignedTo, classification, createdDateTime, determination, lastUpdateDateTime, severity, and status. Example: severity eq 'high' and createdDateTime gt 2021-01-01T00:00:00Z |    No    |
| **Limit**       | The maximum number of incidents to return.                                                                      |    No    |
| **Skip**        | The number of incidents to skip before returning the results.                                                   |    No    |
| **Count Incidents** | Instead of returning all the results, the count parameter returns the number of items in the result set. Default: false |    No    |
