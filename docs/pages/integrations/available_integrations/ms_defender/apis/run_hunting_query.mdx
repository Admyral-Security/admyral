# Run Hunting Query

Run a Kusto Query Language (KQL) query against Microsoft 365 Defender data.

<Callout type="info">
	For Microsoft's documentation, see [Run Hunting
	Query](https://learn.microsoft.com/en-us/graph/api/security-security-runhuntingquery?view=graph-rest-1.0&tabs=http).
</Callout>

## Parameters

| Field        | Description                                                                  | Required |
| ------------ | ---------------------------------------------------------------------------- | :------: |
| **Query**    | The hunting query in Kusto Query Language (KQL).                             |   Yes    |
| **Timespan** | The interval of time over which to query data, in ISO 8601 format. Optional. |    No    |
