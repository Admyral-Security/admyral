# Get Incident

Get the details of an incident.

<Callout type="info">
    For Microsoft's documentation, see [Microsoft Defender Get Incident API](https://learn.microsoft.com/en-us/graph/api/security-incident-get?view=graph-rest-1.0&tabs=http).
</Callout>

## Parameters:

| Field             | Description                                                                                                         | Required |
| ----------------- | ------------------------------------------------------------------------------------------------------------------- | :------: |
| **Incident ID**   | The ID of the incident to retrieve.                                                                                 |   Yes    |
