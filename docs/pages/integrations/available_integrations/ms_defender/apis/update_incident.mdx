# Update Incident

Update the specified properties of an incident.

<Callout type="info">
    For Microsoft's documentation, see [Microsoft Defender Update Incident API](https://learn.microsoft.com/en-us/graph/api/security-incident-update?view=graph-rest-1.0&tabs=http).
</Callout>

## Parameters

| Field           | Description                                               | Required |
| --------------- | --------------------------------------------------------- | :------: |
| **Incident ID** | The ID of the incident to update.                         |   Yes    |
| **Assigned To** | Owner of the incident, or null if no owner is assigned.   |   No     |
| **Classification** | The classification of the incident. Possible values: unknown, falsePositive, truePositive, informationalExpectedActivity, unknownFutureValue. | No |
| **Determination** | The determination of the incident. Possible values: unknown, apt, malware, securityPersonnel, securityTesting, unwantedSoftware, other, multiStagedAttack, compromisedUser, phishing, maliciousUserActivity, notEnoughDataToValidate, confirmedUserActivity, lineOfBusinessApplication, unknownFutureValue. | No |
| **Status** | The status of the incident. Possible values: active, resolved, redirected, unknownFutureValue. | No |
| **Custom Tags** | Array of custom tags associated with an incident.         |   No     |
