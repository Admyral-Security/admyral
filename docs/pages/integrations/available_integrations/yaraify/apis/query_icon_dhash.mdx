# Query icon dhash

Get a list of recent files associated with a specific icon dhash.

<Callout type="info">
	For YARAify's documentation, see [Query icon dhash](https://yaraify.abuse.ch/api/#dhash_icon).
</Callout>

## Parameters:

| Field    | Description                                 | Required |
| -------- | ------------------------------------------- | :------: |
| **icon dhash** | The icon dhash to query for.               |   Yes    |
