# Query a File Hash

Query YARAify for a file hash (SHA256 hash, MD5 hash, SHA1 hash or SHA3-384 hash).

<Callout type="info">
	For YARAify's documentation, see [Query a file
	hash](https://yaraify.abuse.ch/api/#query-filehash).
</Callout>

## Parameters:

| Field    | Description                                 | Required |
| -------- | ------------------------------------------- | :------: |
| **Hash** | File hash of the file you want to download. |   Yes    |
