# Query gimphash

You can get a list of recent files associated with a specific gimphash.

<Callout type="info">
	For YARAify's documentation, see [Query gimphash](https://yaraify.abuse.ch/api/#gimphash).
</Callout>

## Parameters:

| Field    | Description                                 | Required |
| -------- | ------------------------------------------- | :------: |
| **gimphash** | The gimphash to query for.               |   Yes    |
