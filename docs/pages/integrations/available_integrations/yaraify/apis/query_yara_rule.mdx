# Query ClamAV Signature

You can get a list of recent files associated with a specific ClamAV signature.

<Callout type="info">
	For YARAify's documentation, see [Query ClamAV
	Signature](https://yaraify.abuse.ch/api/#clamav).
</Callout>

## Parameters:

| Field                | Description                        | Required |
| -------------------- | ---------------------------------- | :------: |
| **ClamAV Signature** | The ClamAV Signature to query for. |   Yes    |
