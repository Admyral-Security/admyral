# Query tlsh

You can get a list of recent files associated with a specific tlsh.

<Callout type="info">
	For YARAify's documentation, see [Query tlsh](https://yaraify.abuse.ch/api/#tlsh).
</Callout>

## Parameters:

| Field    | Description                                 | Required |
| -------- | ------------------------------------------- | :------: |
| **tlsh** | The tlsh to query for.               |   Yes    |
