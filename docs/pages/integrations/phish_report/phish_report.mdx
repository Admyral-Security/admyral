# Connect Phish Report

1. In **Phish Report**, click the Account email (e.g. chris@admyral.dev) in the top right corner.

2. Copy the API Key.

3. Go to the **Settings** page in **Admyral**. Click on **Create New Credential** in the **Credentials** section. Select **Phish Report** from the dropdown list. Give your credential a name, e.g. `PhishReport`. Paste the token you copied in the previous step into the **API Key** field. Click **Save Credential**.
