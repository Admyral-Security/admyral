# Send message in a chat

Creates a new chat message in a specified chat.

<Callout type="info">
	For Microsoft's documentation, see [Send message in a
	chat](https://learn.microsoft.com/en-us/graph/api/chat-post-messages?view=graph-rest-1.0&tabs=http).
</Callout>

## Parameters:

| Field       | Description                                                                                                 | Required |
| ----------- | ----------------------------------------------------------------------------------------------------------- | :------: |
| **Chat ID** | The ID of the chat where the message will be sent. Example: `19:2da4c29f6d7041eca70b638b43d45437@thread.v2` |   Yes    |
| **Message** | The content of the message to be sent.                                                                      |   Yes    |
