# Run Your First Workflow

Guide on how to create your first workflow within the **[cloud version](www.google.com)** of Admyral.


## Getting Started

### Cloud version

You can create your first workflow within less than 2 minutes: 

1. Create an account or simply sign-in via Github or Google [here](www.google.com)
2. Click on "*Create new workflow*"
3. Choose a workflow from the Workflow library
