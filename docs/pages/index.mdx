# Overview

> Admyral is an open-source, drag-and-drop security workflow builder with integrated case management.

## Core Platform Features

### Workflow builder
* **Workflow Automations**: Intuitively drag and drop workflow actions to build an automation workflow
* **AI Workflows**: Build AI-enhanced workflows using GPT4
* **Run History**: Analyze your workflow run history 
* **Case Management Integration**: Case management specific actions such as "Open Case" or "Update Case" (*coming soon*)
* **Integrations**: Leverage pre-built integrations to speed up your automation efforts (*coming soon*)
